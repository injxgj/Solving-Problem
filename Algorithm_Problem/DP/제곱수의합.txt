제곱수의 합
 

문제 :
      숫자 N을 제곱수의 합으로 표현하고자 할 때, 사용해야 하는 제곱 수의 최소 개수를 출력하는 프로그램을 작성하시오. 
      예를 들어, 숫자 45를 제곱수의 합으로 표현하고자 할 때 필요한 제곱 수의 최소 개수는 2개이며, 이는 다음과 같다.

      45 = 3^2 + 6^2
 

입력 :
      첫 번째 줄에 N이 주어진다. ( 1 ≤ N ≤ 100,000 )  

출력 :
      필요한 제곱 수의 최소 개수를 출력한다.

 

예제 입력 :
      45
예제 출력 :
      2
      
      
// DP 사용
// DP 테이블 정의 : 해당 인덱스를 만들기위한 제곱수의 최소개수를 저장.
                   ( DP[1 ~ N]까지 순서대로 구한다. 38 ~ 44 line이 이에 해당, 만약 DP[i]값이 0일 경우는 1을 저장
                     그 이외의 경우는 j*j가 i보다 작거나 같을 때까지 j값을 올려가면서 가장 최소값일 경우를 DP에 저장한다.) - 명확히 글로 설명을 못하겠네요.. ㅜㅜ 코드를 보고 이해해보세요

#include <stdio.h>
 
int Dp[100001];
 
int main(void) {
 
    int N;
    
    scanf("%d", &N);
 
    for (int i = 1; i <= N; i++)
    {
        for (int j = 1; j*j <= i; j++) {
            if (Dp[i] > Dp[i - j*j] + 1 || Dp[i] == 0)
                Dp[i] = Dp[i - j*j] + 1;
        }
    }
 
    printf("%d\n", Dp[N]);
 
    return 0;
}
